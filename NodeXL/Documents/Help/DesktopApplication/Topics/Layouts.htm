Layouts

Use the Layout menu to select the algorithm to use to lay out the graph. The following layouts are available:

  • Circle: The vertices are equally spaced around the circumference of a circle.
  • Grid: The vertices are equally spaced in a rectangular grid.
  • Fruchterman-Reingold: The Fruchterman-Reingold algorithm is used. This algorithm models vertices as physical bodies that exert attractive and repulsive forces on one another. Unconnected vertices repel each other, while connected vertices attract.
  • Sugiyama: A variant of the Sugiyama layout algorithm is used. For this layout, NodeXL uses the GLEE graph layout engine created by Lev Nachmanson at Microsoft Research.
  • Random: The vertices are placed in random locations.

If you resize a graph it does not get laid out again; it is simply scaled to fit the graph window. If you want to lay out the graph again, use the Layout, Refresh menu item.
