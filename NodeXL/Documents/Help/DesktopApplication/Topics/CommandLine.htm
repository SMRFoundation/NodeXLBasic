Command Line Arguments

If you start NodeXL from a command prompt or a shortcut, you can specify a graph file that the program should open at startup. You can also tell NodeXL to create a graph from data that has been copied to the clipboard. Here are the command line arguments:

[GraphFileName | /c]  [/type:Simple | Pajek]

If NodeXL.exe is not in your Windows path (and by default it is not), you must specify a full path to the program.  This is "C:\Program Files\Microsoft Research\NodeXL\NodeXL.exe" if you installed the program in its default location.

The command line arguments are case-insensitive and can be in any order.

Opening a Graph File

If you want NodeXL to open a graph file at startup, include GraphFileName in the command. The file name must include a full path. The file can be in either the simple or Pajek format. You can specify the format with the optional /type argument, or let NodeXL derive the format from the file name. .txt files are assumed to be in the simple format, and .net files are assumed to be in the Pajek format.

Here are sample command lines for opening a graph file:

  • "C:\Program Files\Microsoft Research\NodeXL\NodeXL.exe" C:\GraphFiles\Simple.txt
  • "C:\Program Files\Microsoft Research\NodeXL\NodeXL.exe" C:\GraphFiles\Simple.txt /type:Simple
  • "C:\Program Files\Microsoft Research\NodeXL\NodeXL.exe" C:\GraphFiles\Pajek.net
  • "C:\Program Files\Microsoft Research\NodeXL\NodeXL.exe" C:\GraphFiles\Pajek.net /type:Pajek

Pasting From the Clipboard

You can tell NodeXL to create a graph from data that has been copied to the clipboard. The data must be in either the simple or Pajek format. You can specify the format with the optional /type argument. If you omit the /type argument when pasting from the clipboard, the simple format is assumed.

Here are sample command lines for creating a graph from data in the clipboard:

  • "C:\Program Files\Microsoft Research\NodeXL\NodeXL.exe" /c
  • "C:\Program Files\Microsoft Research\NodeXL\NodeXL.exe" /c /type:Simple
  • "C:\Program Files\Microsoft Research\NodeXL\NodeXL.exe" /c /type:Pajek
